API Reference
=============

The following pages document the public API of the FLASh library, grouped by
major subpackages. Each page has a brief description followed by module-level
class and function details.

- `Mesh module <api_mesh.html>`__ - core geometry and mesh utilities
- `PDE module <api_pde.html>`__ - PDE models and solvers
- `ROM module <api_rom.html>`__ - reduced-order interpolators
- `Utils module <api_utils.html>`__ - helper routines and communicators
